Command Summary

Command Format
Add add TASK_NAME [starts START_DATE_TIME ends CLOSE_DATE_TIME tag TAG]
Alias alias COMMAND ALIAS
Delete delete INDEX
Complete complete INDEX
Uncomplete uncomplete INDEX
List list
Find find KEYWORD
Update update INDEX [NAME starts START_DATE_TIME ends CLOSE_DATE_TIME tag TAG remove-tag TAG]
Undo undo
Pin pin INDEX
Unpin unpin INDEX
Live Search searchbox
Change Storage change-to PATH_NAME
Help help
Exit exit

Example of some longer commands:

Add

Example:

  • add meeting starts tomorrow 1pm ends tomorrow 3pm
  • add quiz tag cs2102 tag easy

Alias

Example:

  • alias add +
  • alias alias q

Update

Example:

  • update 2 name shopping ends 9pm
  • update 2 tag family remove-tag friends
  • update 3 ends three hours later

Change-to

Example:

  • change-to data/taskmanager.xml
